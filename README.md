# flight-tracker-uruguay
Proyecto educativo que muestra datos de vuelos sobre Uruguay usando API + regex.

[Link al Portfolio de Aprendizaje](https://docs.google.com/document/d/1tkWbLgncFVAp3UUcpVTeh2R2XVHdIGTfmtAtmuBlmdI/edit?usp=sharing)


Instrucciones de como correr el programa: 

1. Clonar repo:   https://github.com/facundevucu/flight-tracker-uruguay.git
2. Instalar depencias: pip install -r requirements.txt
3. Ejecutar el archivo main.py para levantar el frontend con flask: Le damos al boton run o sino ejecutamos este comando python main.py
4. Hacer CTRL + click izq en cualquiera de las urls que aparezcan.
